## 🛠️ 8 Powerful AI Tools

<details>
<summary><b>🔍 1. Fake Job Detector — Analyzes 50+ signals in under 3 seconds</b></summary>
<br/>

Paste any job posting and our AI instantly checks:

- ✅ Company MCA/GST registration verification
- ✅ Salary benchmark against real India market data
- ✅ Contact legitimacy (Gmail = red flag for corporates)
- ✅ Language pattern & scam phrase detection
- ✅ **Trust Score: 0–100** with detailed breakdown

```
Input  → "50 LPA fresher job, apply at hr@gmail.com, deposit ₹5000"
Output → Trust Score: 12/100 ⛔ HIGH RISK — Do NOT apply
```

</details>

<details>
<summary><b>📄 2. Resume ATS Analyzer — Beat the algorithm before a human sees you</b></summary>
<br/>

- Upload PDF or paste plain text
- AI scores your resume against real ATS systems
- Detects missing keywords for your target role
- Actionable suggestions to jump from 43 → 89 score

</details>

<details>
<summary><b>🎤 3. Interview Prep AI — 500+ questions from 50+ Indian companies</b></summary>
<br/>

- Company-specific questions: TCS, Infosys, Wipro, Razorpay, Swiggy, Zomato
- Real-time AI feedback on your answers
- Behavioral + Technical + HR rounds covered

</details>

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/mehergautam/JobVerify.git
cd JobVerify

# Install frontend dependencies
cd client && npm install

# Install backend dependencies
cd ../server && npm install

# Add environment variables
cp .env.example .env
# Add your GROQ_API_KEY and MONGODB_URI

# Run both servers
npm run dev
```

> **Note:** Render free tier has ~30s cold start. First API call may be slightly slow.
